# forge-project-tracker

Forge SDLC workspace for managing project lifecycles.

## Directory Structure

```
projects/        # Individual project directories with project.yaml
.memory/         # Workspace memory store
.preflight/      # Workspace configuration
  config.yaml    # Workspace settings and defaults
```

## Usage

- `/forge:new` -- Create a new project
- `/forge:status` -- Check project status
- `/forge:board` -- View the project board

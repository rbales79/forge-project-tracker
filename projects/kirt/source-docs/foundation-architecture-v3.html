lifecycle_stateactive → deleted_at_source → purged | superseded
embedding_statuspending → embedded | failed | stale
CollectionsSemanticChunks (text) + SlideChunks (PPTX with slide metadata)
Tenant isolationEvery query scoped by tenant_id. API keys bound to tenants.
